API Functions Documentation

* peliculas_idioma: Return quantity of movies in the given language. E.g. app.route/peliculas_idioma/en

* peliculas_duracion: Returns movie runtime and release year. E.g. app.route/peliculas_duracion/Cars

* franquicia: Returns movie quantity and revenues. E.g. app.route/franquicia/toy story collection

* peliculas_pais: Returns quantity of movies produced by a given country. E.g. app.route/peliculas_pais/United States of America

* productoras_exitosas: Returns revenue and quantity of movies of a given company. E.g. app.route/productoras_exitosas/TriStar Pictures

* get_director: Returns director success and list of movies from the given director. E.g. app.route/get_director/James Cameron

* recomendaciones: Returns 5 similar movies from the given one. E.g. app.route/recomendaciones/The Terminator

** Note: app.route = IP or deploy link of the API
